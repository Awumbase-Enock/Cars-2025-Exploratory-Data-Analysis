# Cars 2025: Exploratory Data Analysis

## Overview

This project explores a dataset containing information on vehicle pricing, performance, and technical specifications. Using Python, the analysis investigates how factors such as horsepower, top speed, fuel type, and engine capacity influence vehicle prices and manufacturer positioning.

The project demonstrates the complete data analysis workflow, including data cleaning, exploratory data analysis (EDA), visualization, feature engineering, and insight generation.

---

## Objectives

* Clean and prepare the dataset for analysis
* Analyze vehicle pricing across manufacturers
* Explore relationships between performance metrics and price
* Compare manufacturers based on technical specifications
* Evaluate vehicle value using a custom price-to-horsepower metric
* Identify trends and outliers within the dataset

---

## Dataset Features

The dataset includes information such as:

* Company Name
* Car Model
* Car Price
* Horsepower
* Top Speed
* Acceleration (0–100 km/h)
* Engine Capacity / Battery Capacity
* Fuel Type
* Torque
* Seating Capacity

---

## Tools and Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Data Cleaning

The following preprocessing steps were performed:

* Removed duplicate records
* Identified and handled missing values
* Converted price data to numeric format
* Standardized categorical variables
* Validated numerical columns and investigated outliers

---

## Exploratory Data Analysis

The analysis addressed questions such as:

* Which manufacturers have the highest average vehicle prices?
* How does horsepower influence vehicle price?
* Is there a relationship between horsepower and top speed?
* How does fuel type affect performance?
* Which manufacturers offer the best value for money?

---

## Key Findings

* Vehicle prices vary significantly across manufacturers.
* Horsepower has a positive relationship with vehicle price.
* Horsepower and top speed are strongly correlated.
* Fuel type influences vehicle performance characteristics.
* Higher prices do not always correspond to better value for money.
* Outliers can significantly affect pricing analysis and should be carefully evaluated.
